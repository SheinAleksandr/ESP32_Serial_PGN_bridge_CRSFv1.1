### Требования для компиляции
Для компиляции прошивки в **Arduino IDE** необходимо:

- Установить библиотеку **Alfredo CRSF**
  - Используется для работы с протоколом CRSF
  - Без данной библиотеки проект не компилируется

После установки библиотеки выполните компиляцию и загрузку прошивки в контроллер.

---

### Подключение к AgOpenGPS
После прошивки контроллера:

1. Подключите устройство к приложению **AgOpenGPS**
2. Откройте **Machine Module**
3. Обновите настройки работы актуатора мощности

---

### Параметры Machine Module

Используются следующие пользовательские параметры:

- **User2** — таймер увеличения мощности в секундах

- **User3** — время работы сброса актуатора мощности  
  Значение: `1 × 100 мс`
- **User4** — 2

---

### Примечание
Корректная работа прошивки напрямую зависит
от правильной настройки параметров в **Machine Module AgOpenGPS**.
